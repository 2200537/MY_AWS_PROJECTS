MY-AWS-PROJECT-1

Task:

Setting up an AWS IAM for company staffers to ensure a secure and efficient access control for all Company resources.

Process:

1- Configuring a robust password for IAM user accounts

2- Creating a group (Mode-group) to hold one permission (Read S3bucket only) for all users

3- Creating IAM user (Mode-Staff1) and adding it to the group created.

4- Sign-in with a user account and confirm that users can only access permitted services

5- Create IAM role and attached permission policy to it.

6- Attached the role to an EC2 instance and launch it

7- Created a secrete access key for the IAM user account (Mode-staff1)
